Ethernet AOI (FTP/SFTP ingest)

      

        Second Ethernet (eth1) for an isolated network AOI that pushes files
        directly via FTP/SFTP onto the NVMe (served under the SMB share, covered
        by retention). Everything for this feature is here: "Save + Apply" writes
        the network + ingest settings; the password has its own button below.

Base network (management)

      

        The primary interface (usually eth0) used for the SMB share and this Web
        UI. "Apply Network" changes the LIVE IP immediately via NetworkManager
        — distinct from the eth1 ingest link above, which is set under
        "Ethernet AOI".
